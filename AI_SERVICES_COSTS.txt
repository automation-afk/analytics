  ================================================================================
                            2. VISION AI (FRAME ANALYSIS)
  ================================================================================

  OPENAI GPT-4o-mini (gpt-4o-mini) [ACTIVE]
  -----------------------------------------
  Used for: Frame Analysis (analyzing video screenshots)

  Pricing:
    - Input:  $0.15 per 1M tokens
    - Output: $0.60 per 1M tokens
    - Images: ~85 tokens per image (low detail mode)

  Typical Usage per Video (10-min video, 10s intervals = ~60 frames):
    - 60 frames x 85 tokens = 5,100 image tokens
    - Text prompt: ~200 tokens
    - Output: ~1,500 tokens
    - Total: ~$0.005 per video

  Estimated Cost: ~$0.005 per video


  ANTHROPIC CLAUDE (claude-sonnet-4-20250514) [DISABLED - for reference]
  ----------------------------------------------------------------------
  Previously used for: Frame Analysis

  Pricing:
    - Input:  $3.00 per 1M tokens
    - Images: ~1,600 tokens per image (768x768)

  Previous Cost per Video: ~$0.36 per video (70x more expensive than GPT-4o-mini)


  --------------------------------------------------------------------------------
  TOTAL FRAME ANALYSIS COST: ~$0.005 per video (using GPT-4o-mini)

  ================================================================================
                          5. DATA STORAGE & AUTH
  ================================================================================

  GOOGLE BIGQUERY
  ---------------
  Used for: Storing video metrics, analysis results, channel data

  Pricing:
    - Storage: $0.02 per GB/month
    - Queries: $5.00 per TB processed (first 1TB/month free)

  Typical Usage:
    - Storage: < 1GB for most channels = ~$0.02/month
    - Queries: Well under free tier for typical usage

  Estimated Cost: ~$0-5/month depending on usage


  GOOGLE OAUTH 2.0
  ----------------
  Used for: User authentication

  Pricing: FREE


  SQLITE (Local Database)
  -----------------------
  Used for: Storing transcripts, frame analysis, emotion data locally

  Pricing: FREE (file-based database)


  --------------------------------------------------------------------------------
  TOTAL INFRASTRUCTURE COST: ~$0-5/month

================================================================================
                        COST OPTIMIZATION TIPS
================================================================================

  1. USE GROQ FOR TRANSCRIPTION (FREE)
    - Groq offers free Whisper transcription with generous limits
    - Only falls back to OpenAI if Groq fails

  2. USE GPT-4o-mini FOR FRAME ANALYSIS
    - Already configured! 70x cheaper than Claude for vision
    - Good enough quality for scene detection

  3. BATCH ANALYSIS
    - Run multiple videos overnight to stay within free tiers

  4. SKIP OPTIONAL FEATURES
    - Emotion analysis is optional ($0.02/video saved)
    - Frame analysis is optional ($0.005/video saved)

  5. CACHE RESULTS
    - Analysis results are stored in BigQuery
    - Re-analysis only when needed
